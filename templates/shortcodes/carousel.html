{# Zola shortcode: carousel Usage in markdown: {{< carousel(images="img1.png,img2.png,img3.png", captions="Caption 1,Caption 2,Caption 3") >}} `images` may be a comma-separated string or a JSON-style array literal. We avoid doing `split` in the template to prevent Tera parse issues; instead we expose the raw value in `data-images` and let JS parse it. #} {% if images is defined %} {% set images_value = images %} {% else %} {% set images_value = "" %} {% endif %} {% if captions is defined %} {% set captions_value = captions %} {% else %} {% set captions_value = "" %} {% endif %}
